Entendendo o DOM(Document Object Model)

Dom -> Modelo para documentos, conjunto de objetos dentro do navegador que lhe da acesso aos componentes do navegador

Arvore DOM -> window (Raiz) -> dentro do window temos varios objetos como location(qual pagina), document(documento atual) e history(pra onde vai, onde estava)

dentro de document temos HTML que tem Head e Body

dentro de Head temos meta, title

dentro de Body temos h1, p( que tem Strong), div
