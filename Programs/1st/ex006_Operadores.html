Operadores do JS

Fámilias principais

Aritiméticos -> '+, -, *, /, %, **' % resto da divisão inteira e ** potência do numero (ao quadrado)

Atribuição

Relacionais -> '>, <, >=, <=, ==, !=', ===(retorna se até o tipo é igual), retornam true e false

Lógicos -> '!', '&&', '||' negação, e, ou

ordem dos operadores -> Aritimeticos -> Relacionais -> Lógicos

Ternário -> ? e : (teste, true, false), o que vai acontecer se for vdd, o que vai acontecer ser for falso
